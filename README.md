# Doodler Wordle
Nuxt + Nest wordle

## Build Setup

```bash
# run tests
$ npm run test:client (client only)
$ npm run test:server (server only)
$ npm run test:e2e (end to end server tests)
$ npm run test (all)
$ npm run dev:server
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
